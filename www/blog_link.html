RTB4FREE Web Site Is Off the Ground

January 31, 2015 by Ben

We just got the domain registered, the Amazon instance, and the RTB engine installed on the instance. The web site is in the initial phase of operation. At this point we have the bidder located on the web site. The web site is not yet complete, but we are making great progress. Check back in two weeks and we should have our act together.

What This is All About

This is another open source RTB system. What's different about this one is that all the pieces are here for a full service RTB - the engine, the campaign management and the user console. The RTB is written in JAVA, the adminsitrative parts are written in Ruby on Rails. The system is designed to be very modular and you should be able to stand up your own version of the system without too much trouble.

Although we are providing you with the pieces, you can interchange your database and user interfaces with our components very easily. This is because all communications between the functional modules is done through a well defined, simple publish subscribe user interface.

Components

The compoments of the RTB4FREE system are:

  • RTB bidding engine in JAVA
  • Ruby On Rails Administration System
  • REDIS based publish subscribe network
  • Simple MongoDB based database system.

Play with the RTB system here on this site to learn about RTB, or download the components from Github. The component parts are all modularized on the first page of the web site.

About

RTB4FREE is a complete real time bidding platform that you can use as the basis of your own RTB system. It's open source under the Apache Creative Commons license. Feel free to use the code, we just ask you give attribution to those parts of our system that we use. If you would would like to discuss the system please contact us.
